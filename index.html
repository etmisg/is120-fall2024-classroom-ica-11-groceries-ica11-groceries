JavaScript Introduction

JavaScript is a programming language that is used to make web pages interactive. It is a high-level, interpreted programming language that conforms to the ECMAScript specification. JavaScript has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions.

Basically, JavaScript is a coding language that allows you to add interactive elements to your website

The greatest use of JavaScript is to add interactivity to your website. JavaScript predominantly interacts with the HTML elements on the page, allowing you to manipulate them in various ways. When HTML is sent to a browser, it creates a visual object, called the Document Object Model (DOM). JavaScript can manipulate the DOM, allowing you to change the appearance and content of the page.

The DOM

The Document Object Model (DOM) is a programming interface for web documents. It represents the html page that's rendered by browsers so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. That way, programming languages can connect to the page.

You can check out the DOM by right clicking on the browser window and selecting "Inspect". Try it now!

Within the inspection window, you can see the HTML structure of the page under the "Elements" tab. This is the DOM. Search through the elements to see how the HTML is structured.

After a little while, navigate to the "Console" tab. This is where you can write JavaScript code to interact with the DOM. Let's write your first bits of JavaScript!

You can print something to the console by typing

console.log("My first JavaScript print statement!");

and pressing enter. This is a great way to test your JavaScript code. Now let's print an alert to the page with:

alert("I'm alerting you with this notification");

Implementing JavaScript in our HTML Files

You can write JavaScript directly in the browser console to edit things on the fly. However, it is best practice to write your JavaScript either within your HTML or in a separate file and link it to your HTML file. By writing your JavaScript in separate files, you can reuse your JavaScript code across multiple pages.

JavaScript can be written in the <head> or <body> section of the HTML file using the <script> tags. This is useful when you want to separate your JavaScript from your HTML.

<script>
   alert("Hello, world!");
</script>

External JavaScript is written in a separate file and linked to the HTML file using the <script> tag. This is useful when you want to reuse your JavaScript code across multiple pages.

<script src="script.js"></script>

JavaScript Variables

Variables are containers for storing data values. In JavaScript, you declare a variable with the var, let, or const keyword.

Variables are used to store data values. In the example below, x, y, and z are variables:

var x = 5;
let y = 6;
const z = x + y;

Variables can be named anything you like, but they must follow a few rules:

  • The name must begin with a letter (a-z, A-Z) or an underscore (_).
  • The name can contain letters, numbers, or the underscore character.
  • The name cannot contain spaces.

Declaring Variables

The let keyword is used to declare a variable that can be reassigned, while the const keyword is used to declare a variable that cannot be reassigned.

The value in the name variable can be changed.

let name = "John";

The value in the pi variable cannot be changed

const pi = 3.14;

Data Types

JavaScript has many data types, including strings, numbers, booleans, arrays, and objects. These will be the most common data types you will encounter when working with JavaScript.

  • Strings are used for storing text.
  • Numbers are used for storing numbers.
  • Booleans are used for storing true or false values.
  • Arrays are used for storing collections of data.
  • Objects are used for storing collections of data using key descriptors and values.

Declaring strings:

let name = "John";

Declaring numbers:

let age = 30;
const pi = 3.14;

Declaring booleans:

let isAdult = true;
const buildingIsTall = false;

Arrays

Arrays are used for storing collections of data. They are declared using square brackets [] and can contain any data type, including strings, numbers, booleans, arrays, and objects.

let fruits = ["apple", "banana", "orange"];
Accessing Array Elements

You can access array elements using the index that represents the position of the value. Array indexes start at 0, so the first element in the array has an index of 0, the second element has an index of 1, and so on.

Accessing an array element:

let fruits = ["apple", "banana", "orange"];
let firstFruit = fruits[0]; // apple
let secondFruit = fruits[1]; // banana
Adding and Deleting Array Elements

You can add elements to an array using the array.push() method, which adds an element to the end of the array. You can remove elements from an array using the array.pop() method, which removes the last element from the array.

Adding an element to an array:

let fruits = ["apple", "banana"];
fruits.push("orange");
fruits.push("grape");
// fruits = ["apple", "banana", "orange", "grape"]
fruits.pop();
// fruits = ["apple", "banana", "orange"]

Objects

Objects are used for storing collections of data using key descriptors and values. They are declared using curly brackets `{}` and can contain any data type, including strings, numbers, booleans, arrays, and other objects.

let person = {
   name: "John",
   age: 30,
   isAdult: true
};
Accessing Object Properties

You can access object properties using dot notation or bracket notation. Dot notation is the most common way to access object properties, but bracket notation is useful when the property name is stored in a variable.

Accessing an object property using dot notation, then bracket notation:

let person = {    name: "John",
   age: 30,
   isAdult: true
};
let personName = person.name; // John
let personAge = person["age"]; // 30

Back to the DOM

The DOM renders our HTML page in a way that JavaScript can interact with it. JavaScript can manipulate the DOM, allowing you to change the appearance and content of the page. This is done by selecting elements on the page and changing their properties.

We can access the document object using the document keyword. The document object represents the entire HTML document, and we can use it to access and manipulate the DOM.

Selecting Elements within the DOM

The best way to learn about manipulating the DOM is to select elements on the page and change their properties. You can select elements using the document.getElementById(), document.getElementsByClassName(), and document.getElementsByTagName() methods.

The document.getElementById() method selects an element by its id attribute.

let cardListSection = document.getElementById("card-list-section");

The document.getElementsByClassName() method selects all elements that have that specific class name.

let cardBodyList = document.getElementsByClassName("card-body");

Same goes for the document.getElementsByTagName() method, which selects all elements with that specific tag name.Same goes for the document.getElementsByTagName() method, which selects all elements with that specific tag name.

let divTags = document.getElementsByTagName("div");

Changing Element Properties

Because the DOM represents the HTML page as a tree of elements, you can select elements on the page and change their properties (properties are the attributes of the elements). You can change an elements properties using the `element.property` syntax.

For example, you can change the innerHTML property of an element to change the text inside it.

let cardTitle = document.getElementById("card-title");
cardTitle.innerHTML = "New Card Title";
More Element Property Changes

Some of the many properties you can change on an element include:

  • innerHTML: The text inside the element.
  • style: The style of the element.
  • className: The class name of the element.
  • src: The source of an image element.
  • href: The URL of an anchor element.
Adding and Removing Elements in the DOM

You can add and remove elements from the DOM using the document.createElement(), element.appendChild(), and element.removeChild() methods.

For example, you can create a new element, set its properties, and append it to the page.

let card = document.getElementById("card"); let cardHeader = document.createElement("div"); cardHeader.innerHTML = "New Card"; card.appendChild(cardHeader);
Styling Elements by adding Classes

You can change the style of an element by adding a class to it. You can add a class to an element using the element.className property. You can also add a class to an element using the element.classList.add() method.

let cardBody = document.getElementById("card-body"); // adds one class cardBody.className = "card-body"; // replaces one class with multiple classes cardBody.className = "bg-primary text-white rounded"; // adds one class cardBody.classList.add("border-bottom");

You can remove a class from an element using the element.classList.remove() method.

cardBody.classList.remove("border-bottom");

Conditional Statements

Conditional statements are used to perform different actions based on different conditions. JavaScript has several conditional statements, including if, else, and else if.

The if statement executes a block of code if a specified condition is true.

const userAge = document.getElementById("user-age").value;
if (userAge >= 18) {
   alert("You are an adult");
}

The else statement executes a block of code if the same condition is false.

if (userAge >= 18) {
   alert("You are an adult");
} else {
   alert("You are a minor");
}

The else if statement executes a block of code if the first condition is false and the second condition is true.

if (userAge >= 18) {
   alert("You are an adult");
} else if (userAge >= 13) {
   alert("You are a teenager");
} else {
   alert("You are a child");
}

Functions

Functions are blocks of code that are designed to do one job. Functions are executed when they are called. You can pass data, known as parameters, into a function. Functions can return data as a result.

Functions are declared using the function keyword, followed by the function name and parentheses. The code block is enclosed in curly brackets.

function greetUser() {
   alert("Hello, user!");
}

Anytime you want to call the function, you simply write the function name followed by parentheses and a semi-colon.

greetUser();

Functions can take parameters, which are values that are passed into the function when it is called. The parameters are listed in the parentheses of the function declaration.

function greetUser(name) {
   alert("Hello, " + name + "!");
}

When you call the function, you pass in the value for the parameter.

greetUser("John");

Functions can return data as a result. You can use the return keyword to return a value from a function.

function addNumbers(num1, num2) {
   return num1 + num2;
}
let sum = addNumbers(5, 10);

Loops

Loops are used to execute the same block of code multiple times. JavaScript has several types of loops, including for, for in, for of, while, and do while.

For the rest of the class we will just focus on for of loops. The for of loop is used to iterate over the values of an iterable object, such as an array or a string. The loop will execute a block of code for each value in the iterable object.

Say we are given an array of fruits:

let fruits = ["apple", "banana", "orange"];

We can loop through the array and print each fruit to the console using the for of loop

for (let fruit of fruits) {
   console.log(fruit);
}

We can grab a list of elements from the DOM and loop through them to change their properties. For example, we can change the background color of each element in a list:

let cardBodyList = document.getElementsByClassName("card-body");
for (let cardBody of cardBodyList) {
   cardBody.style.backgroundColor = "lightblue";
}

You can iterate through the keys and values of an Object using the Object.keys() method and the for of loop. For example, we can loop through the keys and values of a person object:

let person = {
   name: "John",
   age: 30,
   isAdult: true
};
for (let key of Object.keys(person)) {
   console.log(key + ": " + person[key]);
}

Events and Listeners

Events are actions that happen on a webpage, such as a user clicking a button or a page loading. You can use JavaScript to respond to these events by adding event listeners to elements on the page.

Event listeners are used to listen for events on elements and execute a block of code when the event occurs. You can add an event listener to an element using the element.addEventListener() method.

For example, you can add a click event listener to a button element:

let button = document.getElementById("button");
button.addEventListener("click", function() {
   alert("Button clicked!");
});

There are many types of events you can listen for, including click, mouseover, mouseout, keydown, keyup, and more. You can find a full list of events on the MDN Web Docs.
